## 错误详情

### 错误类型

`regression_failed` — 复测失败，修复无效

### 错误描述

第一次修复后，玩家 x 坐标可降至 22，比修复前（25）超界更严重。

```
首次失败: x = 25 (超出 28，偏差 -3)
本次复测: x = 22 (超出 28，偏差 -6)

修复后问题反而加重！
```

### 问题定位

检查 commit abc1230 的修复内容，疑似 clamp 参数传递顺序错误：

```javascript
// 错误修复 (abc1230)
player.x = clamp(player.x + moveX * player.speed * delta, 25, canvas.width - 25);
// 但 clamp 函数内部参数顺序可能有问题

// 正确应该是:
player.x = clamp(player.x + moveX * player.speed * delta, 28, canvas.width - 28);
// 并确认碰撞区域计算正确
```
